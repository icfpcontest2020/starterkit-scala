# Scala Starter Kit

All your source files should go into the [src] directory.

## Building

To build locally you need to have [Scala] installed.

To build the JAR:

```shell script
./build.sh
```

## Running

To run the application:

```shell script
./run.sh https://example.com abc123
```

[scala]: https://www.scala-lang.org/
[src]: src
